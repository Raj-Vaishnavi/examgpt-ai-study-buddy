# ExamGPT - AI Study Buddy

ExamGPT is an AI-powered study assistant that converts PPTs and PDFs into simplified notes, exam-ready answers, viva questions, and quizzes.

## Features
- PDF/PPT Upload
- Easy Language Explanation
- 2/5/8 Mark Answers
- Viva Questions
- Quiz Generator

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- AI APIs
